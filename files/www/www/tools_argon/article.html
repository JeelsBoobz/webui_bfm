Cara setup vnstat bandwidth.

vnstat bandwidth adalah alat monitoring bandwidth berbasis CLI (Command Line Interface) yang digunakan untuk melacak penggunaan bandwidth pada antarmuka jaringan di sistem berbasis Linux dan BSD.
copas code dibawah
pkg i -y root-repo
pkg i -y termux-services vnstat vnstati
Restart Termux (mulai ulang Termux) , kemudian buka lagi.
sv-enable vnstat
mkdir -p $HOME/.termux/boot
cd $HOME/.termux/boot
cat > ./vnstat-service <<EOF
#!/data/data/com.termux/files/usr/bin/sh
termux-wake-lock
sv up vnstat
EOF
su -c 'cat > /data/adb/service.d/vnstat.sh <<EOF
#!/system/bin/sh

(
    until [ $(getprop init.svc.bootanim) = "stopped" ]; do
        sleep 10
    done
    
    PATH=/data/data/com.termux/files/usr/bin
    LD_PRELOAD=/data/data/com.termux/files/usr/lib/libtermux-exec.so
    PREFIX=/data/data/com.termux/files/usr
    
    vnstatd -d
)&
EOF'
su -c 'chmod +x /data/adb/service.d/vnstat.sh'
Selesai, diperlukan reboot.

Cara setup Tailscale Magisk.

Tailscale Magisk cocok untuk akses jarak jauh ke perangkat pribadi, atau membuat jaringan internal tanpa perlu setup VPN manual.
Install module Tailscale Magisk
setelah proses instalasi selesai, reboot device
tunggu beberapa saat biar service Tailscalenya running, kemudian buka Termux
copas code dibawah
su -c tailscale login
lalu muncul berupa link.
copy link tersebut, kemudian paste di browser dan lanjut login
setelah login selesai balik ke Termux
jika ada log di Termux success maka berhasil.
kalau tidak ada, terkadang harus install ulang lagi modulenya
Selesai.

install app SoftApHelper

Cara buat auto enable hotspot

1. Untuk android 10 keatas
* Opsi pertama tanpa perlu Termux:boot
copas code dibawah
su -c 'cat > /data/adb/service.d/hotspot.sh <<EOF
#!/system/bin/sh
until [ "$(getprop init.svc.bootanim)" = "stopped" ]; do
    sleep 30
done

su -c 'service call connectivity 24 i32 0 i32 0 i32 0 s16 random && service call tethering 4 null s16 random'
EOF'
su -c 'chmod +x /data/adb/service.d/hotspot.sh'
* Opsi kedua memerlukan Termux:boot
copas code dibawah
mkdir -p $HOME/.termux/boot
cd $HOME/.termux/boot
cat > ./hotspot-service.sh <<EOF
#!/system/bin/sh
until [ "$(getprop init.svc.bootanim)" = "stopped" ]; do
    sleep 30
done

su -c 'service call connectivity 24 i32 0 i32 0 i32 0 s16 random && service call tethering 4 null s16 random'
EOF
2. Untuk semua versi android, tetapi harus menyesuaikan sesuai dengan tempat hotspot
* Opsi pertama tanpa perlu Termux:boot
copas code dibawah
su -c 'cat > /data/adb/service.d/hotspot.sh <<EOF
#!/system/bin/sh
sleep 30
am start -n com.android.settings/.TetherSettings
input keyevent 66
input keyevent 66
input keyevent 66
input key event 3
EOF'
su -c 'chmod +x /data/adb/service.d/hotspot.sh'
* Opsi kedua memerlukan Termux:boot
